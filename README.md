# Discord SE - 저사양 컴퓨터용 디스코드
## Super Light Discord
![](image/icon.png)

## 기능

- 메모리 점유율 감소
- CPU 점유율 감소
- 속도 증가

## 장단점
### 장점
- 저사양 컴퓨터에서 원활한 구동과 라이브, 음성채팅 가능

### 단점
- 마이크,카메라 등등이 default로 설정되어있고 바꿀수 없음
